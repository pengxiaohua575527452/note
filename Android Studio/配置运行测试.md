# 如何在 Android Studio 中添加一个运行配置，用来运行指定的单元测试

## 概述

- Kotlin/Js 工程
- 通过添加 Gradle 运行配置实现
- 以 jsNodeTest 运行
- 项目配置
  - build.gradle.kts
  ```
    plugin {
      kotlin("mutiplatform") version "1.9.20"
    }

    repositories {
        mavenCentral()
    }

    kotlin {
      js(IR){
        nodejs
      }
    }

  ```
